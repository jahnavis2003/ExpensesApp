# My Node.js Application

This is a Node.js backend application built with Express and Mongoose.

## 🚀 Features

- User authentication
- Expense tracking
- MongoDB database with Mongoose
- RESTful API structure

## 🛠️ Tech Stack

- Node.js
- Express
- MongoDB & Mongoose
- JWT (for auth)
- dotenv (for environment config)

## 📦 Setup

```bash
npm install
cp .env.example .env
npm start
